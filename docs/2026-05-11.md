# 2026-05-11 개발 진행 보고서

## 프로젝트 개요
**PetTogether** (구 Bemypet) — 반려동물 커뮤니티 웹/모바일 앱

- **라이브 URL**: https://bemypet-clone.vercel.app
- **GitHub**: https://github.com/eunji020425-rgb/bemypet-clone
- **기술 스택**: Next.js 16 · TypeScript · Tailwind CSS · Supabase · Vercel · Capacitor · Leaflet · Gemini 2.0 Flash · Naver Local Search API

---

## 오늘 완료한 작업

### 1. 프로젝트 초기 세팅
- Next.js 16 (App Router, TypeScript, Tailwind) 프로젝트 생성
- Supabase 클라이언트 설정 (`@supabase/ssr`)
- 환경변수 파일 구성
- Capacitor 6 모바일 앱 패키지 설치

### 2. 데이터베이스 구축 (Supabase)
**테이블:**
- `profiles` — 사용자 프로필 (auth.users 자동 연동 트리거)
- `posts` — 자유게시판 게시글
- `post_likes` — 좋아요 (UNIQUE 제약)
- `comments` — 댓글
- `chat_messages` — 실시간 채팅 메시지

**보안/기능:**
- 모든 테이블 RLS(Row Level Security) 정책 적용
- `chat_messages` Realtime 구독 활성화
- Storage `images` 버킷 생성 (이미지 업로드용)

### 3. 인증 시스템
- 이메일 회원가입/로그인
- Google OAuth 로그인 (Supabase Auth + Google Cloud Console 연동)
- OAuth 콜백 라우트 (`/auth/callback`)
- 미들웨어 기반 세션 관리 (Next.js 16의 `proxy.ts` 컨벤션 적용)

### 4. 자유게시판 (커뮤니티)
- 게시글 목록 페이지 (`/community`)
- 글쓰기 페이지 (`/community/write`) — 이미지 첨부 지원
- 게시글 상세 페이지 (`/community/[id]`)
- 좋아요 기능 (낙관적 업데이트)
- 댓글 작성/조회

### 5. 실시간 채팅 (`/chat`)
- Supabase Realtime postgres_changes 구독
- 메시지 전송 시 다른 사용자에게 즉시 반영
- 카카오톡 스타일 UI (내 메시지 우측 / 상대 메시지 좌측)
- 자동 스크롤 하단 고정

### 6. 동물병원 찾기 (`/hospital`) — 신규
- 현재 위치 기반 (브라우저 Geolocation API)
- **Naver Local Search API** 연동 (`/api/hospitals`)
- **Leaflet + OpenStreetMap** 지도 표시
- 병원 핀 클릭 → 상세 정보 팝업
- 거리순 정렬 (Haversine 공식)
- 위치 권한 상태 표시 (locating/success/denied/fallback) + 재시도 버튼

### 7. AI 닥터 (`/ai-doctor`) — 신규
- **Gemini 2.0 Flash** 연동 (수의사 시스템 프롬프트)
- 빠른 질문 템플릿 6개 제공
- 스트리밍 응답 (글자가 실시간 타이핑되듯 출력)
- Edge Runtime 적용 (전세계 가까운 서버에서 실행)
- 대화 컨텍스트 유지 (최근 6턴)

### 8. 산책로 추천 (`/walk`) — 신규
- **Gemini + 위치 정보** 조합으로 AI 추천
- JSON 응답 모드 (`responseMimeType: 'application/json'`)
- 반려동물 종류 선택 (강아지/고양이/소형견/대형견/노령견)
- 지도에 산책로 핀 표시
- 카드 클릭 시 상세 정보(특징, 팁) 펼침
- 난이도별 색상 구분

### 9. 디자인 / 브랜딩
- 사이트명: Bemypet → **PetTogether** 변경
- 컬러: 크림 배경(#fdf8f0) + 노란 포인트(#f5c518)
- 헤더 네비게이션: 홈, 커뮤니티, 실시간채팅, 동물병원, AI닥터, 산책로
- 반응형 디자인 (모바일 메뉴 포함)

### 10. 성능 최적화
- `vercel.json`에 서울 리전(`icn1`) 설정
- 정적 자산 캐시 헤더 (1년 immutable)
- 커뮤니티/채팅 페이지 스켈레톤 로딩 UI
- API 라우트 Edge Runtime 변환
- Gemini 응답 스트리밍
- 메시지 히스토리 슬라이싱(컨텍스트 최소화)

### 11. 배포

## 사용 중인 외부 API

| 서비스 | 용도 | 키 종류 |
|--------|------|---------|
| Supabase | DB, Auth, Storage, Realtime | URL + anon key |
| Naver Developers | 지역(병원) 검색 | Client ID + Secret |
| Google Gemini | AI 닥터, 산책로 추천 | API Key |
| OpenStreetMap | 지도 타일 | 무료 (API 키 불필요) |

## 해결한 주요 이슈

1. **Next.js 16의 middleware → proxy 컨벤션 변경 적용**
2. **Supabase placeholder URL 빌드 실패** → 가드 로직 추가
3. **새 `sb_publishable_*` 키 다른 컴퓨터에서 차단** → 레거시 anon key로 교체
4. **Vercel 미국 리전 배포 → 한국 사용자 레이턴시** → `regions: ["icn1"]` 설정
5. **Geolocation 무음 실패** → 상태 표시 + 재시도 UI 추가
